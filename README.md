# Airline Revenue Calculator

## Setup
1. Install dependencies:
   pip install -r requirements.txt

2. Run app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000/

## Features
- Calculates airline revenue based on passenger counts, distance, and aircraft type.
- Surcharges, discounts, and bonuses applied.
- Simple form input and result display.

## Project Structure
- app.py
- templates/index.html, result.html
- static/style.css
- requirements.txt
